Podstawowe znaczniki HTML

<!DOCTYPE HTML>

Informuje przeglądarkę o wykorzystaniu HTML5 na stronie.

<html> </html>

<html lang = "pl"> </html>

Rozpoczyna kod html.

Atrybut lang informuje o języku na stronie.

<head> </head>

Zawiera elementy informacyjne, niewidoczne na stronie.

<script> </script>

<script src = "code.js"> </script>

Zapisuje się w nim linie skryptu.

Z atrybutem src może też służyć do podpięcia pliku zewnętrznego.

<body> </body>

Zawiera elementy widoczne na stronie.

<div> </div>

Element blokowy, mogący przechowywać inne elementy strony. Posiada fizyczne wymiary, takie jak np. szerokość i wysokość.

<span> </span>

Element separujący, który nie posiada fizycznych wymiarów i dlatego nie ingeruje w układ elementów blokowych na stronie.

<a> Tekst wyświetlany jako link </a>

<a href = "#"> </a>

Znacznik hiperłącza.

Do atrybutu href wpisujemy adres docelowy. Zamiast adresu można podać #, co spowoduje powrócenie do góry strony, albo ID znacznika poprzedzony #

<h1> Tekst nagłówkowy </h1>

...

<h6> Tekst nagłówkowy </h6>

Znacznik nagłówna. Tekst zamknięty w środku będzie miał większy rozmiar niż zwyczajny tekst i będzie on malał wraz ze zwiększaniem się cyfry w znaczniku.

<p> Treść akapitu </p>

Akapit tekstu.

<img src = "image.png" alt = "Image">

Znakcznik obrazu.

Do atrybutu src wpisujemy scieżkę do pliku. Natomiast w atrybucie alt ustawiamy tekst, wyświetlany, gdy nie będzie można załadować obrazu.

<br>

Znacznik końca lini.

<i> </i>

Znacznik kursywy.

<b> </b>

Znacznik pogrubiający czcionkę.

<HEAD> Metainformacje </HEAD>

<meta charset = "utf-8">

Ustawia kodowanie znaków.

<meta name = "author" content = "Jan Kowalski">

Podaje informacje o autorze strony.

<meta name = "description" content = "Krótki opis strony">

Zawiera opis strony, wyświetlany w wyszukiwarce.

<meta name = "keywords" content = "naj, strona, ever">

Zawiera słowa kluczowe, pomagające w pozycjonowaniu strony w wyszukiwarce.

<title> </title>

Tytuł wyświetlany na górze karty, obok faviconu.

<HEAD> Linki </HEAD>

<link rel = "stylesheet" href = "style.css">

Podpina arkusz stylów do strony.

<link rel = "icon" href = "favicon.png">

Ustawia małą ikonę przy tytule strony.

<BODY> HEADER </BODY>

<header> </header>

Opisuje nagłówek strony.

<nav> </nav>

Służy do opisywania nawigacji (menu) strony.

<BODY> MAIN </BODY>

<main> </main>

Opisuje główny element strony.

<section> </section>

Używany jest do dzinelenia elementu main na mniejsze sekcje.

<article> </article>

Używany jest do dzinelenia elementu main lub section na mniejsze sekcje.

<BODY> FOOTER </BODY>

<footer> </footer>

Opisuje stopke na stronie.

Znaki specjalne

&quot;

Wyświetla znak ".

&amp;

Wyświetla znak &.

&lt;

Wyświetla znak <.

&gt;

Wyświetla znak >.

Wyświetla znak © (copyright).

Selektory CSS

*

{

margin: 0;

padding: 0;

}

Tak jak podano na przykładzie po lewej * jest selektorem wszystkich elementów na stronie.

Na samym początku pisania strony warto wyzerować marginesy wszystkim obiektom na stronie.

div

{

background-color: black;

}

W tym przypadku div jest selektorem znacznika html a wszystko pomiędzy klamrami deklaracją.

#menu

{

background-color: black;

}

W tym przypadku #menu jest selektorem ID.

.button

{

background-color: black;

}

W tym przypadku .button jest selektorem klasy.

div, #menu, .button

{

background-color: black;

}

Teraz div, #menu i .button będą przyjmować takie same właściwości.

div p

{

text-align: center;

}

Wszystkie p znajdujące się w jakimkolwiek znaczniku div przjmą wartość text-align: center;

div > p

{

text-align: center;

}

Wszystkie p znajdujące się bezpośrednio w znaczniku div przjmą wartość text-align: center;

input [type = "text"]

{

text-align: center;

}

Podane wartości przyjmą jednynie takie znaczniki input, które posiadją atrybut type o wartości text.

CSS - Kolory - CSS

red

cyan

aqua

Kolor opisywany słownie.

#ff0000

#f00

#333333

#333

Kolor opisywany szesnastkowo. Pierwsze dwie cyfry po znaku # to kolor czerwony, kolejne dwie to zielony i ostatnie to niebieski.

Jeśli obie cyfry każdego koloru się powtarzają można zapisać tylko jedną z nich;

rgb(255, 0, 0)

rgba(255, 0, 0, 1)

rgba(255, 0, 0, 0.5)

Pierwsza wartość to czerwony, druga to zielony a trzecia to niebieski. Każdy kolor przyjmuje wartości od 0 do 255.

rgba() posiada czwartą wartość, którą jest przeźroczystość. 0 oznacza całkowicie przeźroczysty a 1 w pełni widoczny.

1 in = 72 pt

1 pc = 12 pt

1 vw

1 vmax

CSS - Tło - CSS

background-color: red;

background-color: #ff0000;

background-color: rgb(255, 0, 0);

Zmienia kolor tła.

background-image: url("bg.png");

background-position: left top;

background-repeat: no-repeat;

background-repeat: repeat-x;

background-repeat: repeat-y;

background-image ustawia podany obraz jako tło strony.

background-position ustawia, pozycję tła.

background-repeat ustawia czy i jak ma powtarzać się obraz.

background: linear-gradient(45deg, silver, black);

background: radial-gradient(circle, silver, black);

Ustawia gradient jako tło strony.

opacity: 1;

Ustawia przeźroczystość.

0 oznacza całkowitą przeźroczystość, 1 w pełni widoczny obiekt.

CSS - Obiekty blokowe - CSS

display: block;

display: inline-block;

display: flex;

Zmienia fizyczne zachowanie elementu na stronie.

width: 10px;

width: 10em;

width: 10vh;

width: 10vw;

width: 10%;


min-width: 10%;

max-width: 10%;

Zmienia szerokość obiektu. Nie trzeba podawać statycznej szerokości, możemy określić minimalny oraz maksymalny zakres.

height: 10px;

height: 10em;

height: 10vh;

height: 10vw;

height: 10%;


min-height: 10%;

max-height: 10%;

Zmienia wysokość obiektu. Nie trzeba podawać statycznej wysokości, możemy określić minimalny oraz maksymalny zakres.

position: static;

position: relative;

position: absolute;

position: fixed;

Pozycja statyczna jest domyślną pozycją, w której obiekt ustawia się w odniesieniu do innych elementów blokowych.

Pozycja relatywna sprawia, że obietk utawia się w odniesieniu do swojej pozycji.

Pozycja absolutna ustawia obietk w odniesieniu do nadrzędnego obiektu relatywnego lub jeśli takiego nie ma, do maksymalnego lewego, górnego punktu strony.

Pozycja fixed "przykleja" obiekt do danego miejsca na ekranie i nie zmienia go, pomimo scrollowania strony.

top: 10vh;

right: 10vw;

bottom: 10vh;

left: 10vw;

Ustawiają odległość od danej krawędzi obiektu nadrzędnego.

Opcje te nie działają ze statyczną pozycją!

margin-top: 10vh;

margin-right: 10vw;

margin-bottom: 10vh;

margin-left: 10vw;


margin: 10vh;

margin: 10vh 10vw;

margin: 10vh 10vw 10vh;

margin: 10vh 10vw 10vh 10vw;

Są to marginesy zewnętrzne. Ustawiają odległość od danej krawędzi obiektu nadrzędnego.

Przy wpisaniu tylko jednej wartości we właściwości margin, przyjmują ją wszystkie cztery marginesy;

Przy wpisaniu dwóch wartości we właściwości margin, pierwsza oznacza marginesy wertykalne a druga marginesy horyzontalne;

Przy wpisaniu trzech wartości we właściwości margin, pierwsza oznacza margines górny, druga marginesy poziome, a trzecia margines dolny;

Przy wpisaniu czterech wartości we właściwości margin, pierwsza oznacza margines górny, druga margines prawy, trzecia margines dolny, a czwarta margines lewy;

padding-top: 10vh;

padding: 10vh 10vw;

padding: 10vh 10vw 10vh 10vw;

Padding to margines wewnętrzny. Określa odległość elementów w środku obiektu od jego krawędzi.

Kolejność podawania wartości jest taka sama jak we właściwości margin (z ruchem wskazówek zegara).

border: 2px solid #fff;

border: 2px dashed #fff;

border: 2px dotted #fff;

border: 2px double #fff;


border: 2px groove #fff;

border: 2px ridge #fff;

border: 2px inset #fff;

border: 2px outset #fff;

Obramowanie elementu blokowego.

Tak jak w margin oraz padding możemy wyodrębnić części obramowania (border-top, border-left, itp.).

Przy skróconym zapisie (na przykładach) nie można pominąć stylu obramowania (solid, double, groove, itp.)

Pierwsze cztery style przedstawiają obramowanie "2D" a kolejne "3D".

box-shadow: 5px 5px 5px gray;

Cień obiektu.

Pierwsza wartość oznacza przesunięcie w poziomie, druga w pionie, trzecia rozmycie, a czwarta kolor.

box-sizing: border-box;

box-sizing: content-box;

border-box wlicza marginesy, paddingi, itp. w wymiary elementu blokowego.

content-box natomiast dolicza marginesy, paddingi, itp. do wymiarów elementu blokowego.

float: left;

float: right;

Właściwość ta sprawia, że obiekt przylega w poziomie do podanej krawędzi elementu nadrzędnego lub sąsiedniego.

clear: left;

clear: right;

clear: both;

Zamyka obszar działania właściwości float.

Po zakończeniu ustawiania obiektów warto dodać na końcu element np. z clear: both;

font-style: italic;

font-style: oblique;

font-variant: small-caps;

text-align: left;

text-align: right;

text-align: center;

text-align: justify;

Justuje tekst.

text-decoration: line-through;

text-decoration: overline;

text-decoration: underline;

text-indent: 10px;

text-transform: capitalize;

text-transform: lowercase;

Tranformuje tekst.

color: white;

color: #fff;
